## Tips for Best Results

### Midjourney
- Use `/imagine` command and paste each prompt
- Add `--ar 16:9 --s 750 --q 2` for cinematic widescreen
- Upscale your favorite variation to max resolution
- Download at highest quality

### DALL-E 3 (ChatGPT)
- Paste the prompt without the `--ar` flags
- Select "Landscape" orientation
- Select "HD" quality
- You may need to simplify longer prompts slightly

### Leonardo AI (Free Option)
- **Phoenix** — best all-rounder, high detail (recommended)
- **Kino XL** — cinematic style, dramatic lighting (great for this project)
- **PhotoReal V2** — lifelike realism (needs Alchemy toggle ON)
- **DreamShaper V7** — fantasy/mystical scenes
- Set dimensions to 1920x1080
- Guidance scale: 7-9
- Generate 4 variations, pick the best
- Remove the `--ar 16:9 --s 750 --q 2` flags (those are Midjourney-only)

### Consistency Tips
- Generate all 20 images in the same session if possible
- Use consistent style keywords across all prompts
- For Midjourney: add `--sref [URL]` with a reference image to keep style consistent
- Adam and Eve should look consistent across scenes — describe them the same way
